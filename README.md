# Ryan's Grimoire Blog Hexo Workspace

## Get Started

### Create a New Post

```shell
hexo new "<Title>"
```

### Run Server Locally

```shell
hexo server
```

### Draft a Post

```shell
hexo new draft "<Title>"
```

### Show Drafts When Running Server Locally

```shell
hexo server --draft
```

### Publish a Draft

```shell
hexo publish "<Title>"
```

### Remove Unused Tags

```shell
hexo clean
```
